# Polibook
This program implements WebGL in a 2D polibook .

The application has three modes: File, Draw, and Paint
### File Mode
File Mode (default): enter File Mode by pressing the 'f' key
### Draw Mode
Draw Mode: enter Draw Mode by pressing the 'd' key
### Paint Mode
Paint Mode (Extra): enter Paint Mode by pressing the 'p' key,
 press number one through nine to select brush size, or you can input the brush size in its input box.
